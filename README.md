Healthcare Agent – Basic Symptom Analysis (Non-Diagnostic)
This project implements a Healthcare Assistant using Google AI Studio.
The agent provides basic symptom discussion and general wellness guidance
for educational purposes only.

Disclaimer:
This is not medical advice. Always consult a qualified healthcare professional.

Goal
- Perform basic symptom analysis (non-diagnostic)
- Ask clarifying questions before giving suggestions
- Provide general wellness advice only
- Escalate to emergency care for severe or sudden symptoms

 Agent Rules
- Does not diagnose or name diseases and does not reveal internal reasoning or analysis
- Always asks clarifying questions
- Provides only user-facing responses

Example Interaction between AI agent and concerned individual (USER)

User: I have a headache.

Agent:
- Starts with a disclaimer
- Asks about duration and severity
- Suggests rest and hydration
- Advises seeing a professional if symptoms worsen


  Platform used for Ai agnet creation
Platform: Google AI Studio
Model: Gemini (via AI Studio)
Prompt Type: System instruction-based healthcare assistant
